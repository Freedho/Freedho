Selamat datang di portofolio saya, saya Fridho Yudhistira. Saya adalah seorang pengembang web yang memiliki minat dan hobi yang beragam. Di luar pekerjaan saya, saya suka bermain game, membaca manga, menonton anime, dan menonton drakor. Saya percaya bahwa hobi dan minat saya dapat membantu saya dalam mengembangkan kreativitas dan kemampuan saya dalam pengembangan web. Saya juga percaya bahwa dengan memiliki hobi dan minat yang beragam, saya dapat memiliki perspektif yang lebih luas dan dapat menyelesaikan masalah dengan cara yang lebih kreatif. Dalam portofolio ini, saya akan membagikan beberapa proyek yang telah saya kerjakan, serta beberapa informasi tentang diri saya dan pengalaman saya dalam pengembangan web. Saya harap Anda dapat menemukan informasi yang berguna dan menarik dalam portofolio ini. Terima kasih atas kunjungan Anda, dan saya berharap Anda dapat menikmati pengalaman menjelajahi portofolio saya.
Saya bercita-cita ingin menjadi Fullstack Web Developer, saya yang memiliki pengalaman dalam pengembangan web selama 2 tahun dimulai dari kelas 10. Saya memiliki kemampuan dalam menggunakan bahasa pemrograman seperti HTML, CSS, dan C++, serta framework seperti Bootstrap. Saya juga memiliki pengalaman dalam menggunakan database seperti MySQL, serta memiliki kemampuan dalam mengembangkan aplikasi web yang responsif dan dapat diakses oleh pengguna dengan berbagai perangkat.
